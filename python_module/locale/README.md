locale
******

This directory holds subdirectories for each supported translation,
holding the translated strings required for use with gettext.

Layout
======
An example layout is shown below:

siril/
├── locale/           # Translations directory
│   ├── es/
│   │   └── LC_MESSAGES/
│   │       └── siril.mo
│   └── fr/
│       └── LC_MESSAGES/
│           └── siril.mo
